证券投资分析的主要方法

有效市场假说

只要证券的市场价格充分及时反映了全部有价值的信息，且市场价格代表着证券的真实价格，这样的市场就被称为有效市场。有效市场分为三类：

  • 弱型（weak form）有效市场
  • 半强型（semi-strong form）有效市场
  • 强型（strong form）有效市场

若市场现在价格充分反映了历史信息，这种市场即为弱型有效市场。在弱型有效市场中投资者不可能通过对股票历史的分析而获得超额利润，因为股价已经根据这些信息做了相应调整。若市场现在价格不仅反映了历史信息，而且反映了所有与公司证券有关的公开信息，这种市场即为半强型有效市场。若有关股票的公开信息对其价格的变动仍有影响，则这样的市场尚未达到半强型有效市场，在半强型有效市场中，对一家公司的资产负债表、损益表等可公开获得的信息进行分析均不可能获得超额利润。若市场价格充分反映了有关公司的一切可得信息，从而使任何获得内幕消息的人也不能凭此获得超额利润时，这样的市场即为强型有效市场。强型有效市场中，市场价格不仅反映了一切历史信息和一切可公开获得的信息，同时也充分反映了一切内幕信息。

根据市场类型不同，可以知采取不同分析方法分析股票价格。若市场未达到弱型有效市场，则通过对历史信息分析出其未来价格；若市场未达到半强型有效市场，则分析公开资料能增加收益；若市场达到强有效市场，则任何企图寻找内部资料信息来打击市场的做法都是不明智的，对于证券组合管理者来说，此时会选择消极保守型的态度，只求获得市场的平均收益水平。

证券投资的基本分析方法

证券分析主要有基本分析和技术分析两大类。这两者区别在于基本分析立足于未来分析股份变动趋势；技术分析则着眼于过去，用历史数据和变化规律预测股份趋势。基本分析主要解决投资对象问题，即投资何种股票；技术分析主要解决投资时机问题，即解决何时买卖股票问题；基本分析着重分析股市的长期变动趋势，而技术分析着重分析股市的中短期变动趋势。

基本分析

基本分析又称为基本面分析，其从宏观经济、行业、区域和公司等方面对价格做分析。

技术分析

技术分析是指仅从证券的市场行为来分析证券价格未来变化趋势的方法。证券的市场行为有多种表现形式，如市场价格、成交量等。技术分析是建立在市场行为包含一切信息、价格沿趋势移动和历史会重复这三个假设之上的。粗略的分，可将技术分析理论分为K线理论、切线理论、形态理论、技术指标理论、波浪理论和循环周期理论等。
